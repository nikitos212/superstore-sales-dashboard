# Superstore Sales Dashboard

Интерактивный BI-дашборд для анализа продаж интернет-магазина.

## Быстрый старт

1. Установите PostgreSQL
2. Выполните SQL-скрипты из папки `sql/`
3. Откройте `powerbi/superstore_dashboard.pbix` в Power BI Desktop
4. Обновите источник данных

## Ключевые метрики

- **Выручка**: $2.3M
- **Прибыль**: $300K
- **Заказы**: 9,994
- **Клиенты**: 793
- **Маржа**: 13%

## Технологии

- **База данных**: PostgreSQL
- **BI инструмент**: Power BI
- **Языки**: SQL, DAX
- **Визуализация**: 8 интерактивных графиков
